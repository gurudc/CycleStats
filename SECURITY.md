# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest (main) | ✅ |

## Reporting a Vulnerability

This is a self-hosted personal project. If you find a security issue, please open a [GitHub Issue](https://github.com/gurudc/CycleStats/issues) or contact the maintainer directly.

For sensitive disclosures, please use GitHub's private vulnerability reporting feature.
